Feature Set

For the feature set that I used, there were a few considerations to make as noted in the Visualizations section. Some variables, like paid_total, provided information that someone trying to determine an interest rate wouldn't have. Others, like sub_grade, were too close to just outright stating the interest rate. Still others, like current_accounts_delinq, provided so little information that including them would pointlessly bloat the machine learning algorithm. Finally, many variables needed to make small changes to their data in order to properly fit in a feature set.

Here's a list of changes made from the original data set to the feature set:

  • Removed from set: installment, grade, sub_grade, issue_month, loan_status, initial_listing_status, disbursement_method, balance, paid_total, paid_principal, paid_interest, paid_late_fees
  • Removed for having too little information: current_accounts_delinq, num_accounts_30d_past_due
  • Changed to integer values: emp_title, state, homeownership, verified_income, verification_income_joint, loan_purpose, application_type
  • Changed NA/blank spaces to appropriate values:emp_title, verification_income_joint, emp_length, annual_income, annual_income_joint, debt_to_income_joint, months_since_last_delinq, months_since_90d_late, months_since_last_credit_inquiry, num_accounts_120d_past_due

Algorithms

For the first algorithm, I decided to use the relatively simple linear regression algorithm. Since every input variable and the output in my feature set was a number, a linear regression made some sense. It may lack the "intelligence" of a more advanced learning algorithm, but the low amount of processing power required to generate and use the regression could potentially offset the algorithm's relative lack of precision.

For the second algorithm, I decided to use a neural network with 2 hidden layers. One of the downsides of a linear regression in this case is that many variables aren't particularly well defined by linear relationships. In addition, the importance of many variables strongly depends on others. Take income_joint and application_type, for example - if the loan applicant isn't taking a joint loan, their joint income should play little to no role in their final interest rate. With a neural network, I'd be able to remedy these issues more easily than with the linear regression.
